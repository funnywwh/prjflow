# v0.5.0 发布说明

**发布日期**: 2025年12月05日

## 主要更新

### 1. 指派功能 🎯
- 需求、任务、Bug支持指派，可添加备注和修改状态
- 自动状态修改：需求（draft/reviewing → active）、任务（wait → doing）
- 完整记录指派人变更和状态变更到历史记录

### 2. 个人工作台 🏠
- 支持卡片和Tab的显示/隐藏和排序
- 资源分配列表：工时统计、周期筛选（本周/本月/全部）
- 工作报告列表：展示日报和周报

### 3. 权限控制 🔐
- 按钮级权限：`v-permission` 指令和 `PermissionButton` 组件
- 主要页面全面应用权限控制

### 4. 实体转换 🔄
- 需求、任务、Bug之间支持相互转换，自动关联原实体

### 5. 组件优化 📋
- 项目、需求、任务详情组件封装，减少代码重复
- 负责人选择使用项目成员选择组件，显示角色信息

### 6. 微信登录优化 🔐
- 用户名格式：`{OpenID后8位}_{YYYYMMDD}`
- 权限控制：只有拥有 `user:create` 权限的用户才能添加用户
- 不再自动创建用户，需管理员先添加

## 主要修复

1. ✅ 修复Bug统计页面交互，支持快速跳转到筛选后的列表
2. ✅ 修复需求、任务搜索条件的状态选项
3. ✅ 修复资源分配列表查询问题，优化权限过滤逻辑
4. ✅ 修复微信登录用户名生成逻辑
5. ✅ 修复软删除用户导致创建用户失败的问题
6. ✅ 删除用户改为硬删除

## 升级说明

- **无需数据库迁移**：本次更新不涉及数据库结构变更
- **前端更新**：需要重新构建前端
- **后端更新**：需要重新编译后端
