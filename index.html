快速部署命令

使用 Docker Compose

# 克隆项目
git clone https://github.com/funnywwh/prjflow.git
cd prjflow

# 一键启动
docker-compose up -d

# 访问应用
open http://localhost:8080

传统部署方式

# 启动后端
cd backend && go run cmd/server/main.go

# 启动前端 (开发模式)
cd frontend && npm install && npm run dev
